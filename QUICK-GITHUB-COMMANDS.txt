========================================
RELIABLEBOSTON.COM - QUICK GITHUB SETUP
Auto-Deploy on Git Push
========================================

STEP 1: CREATE GITHUB REPO
Go to: https://github.com/new
Repository name: reliableboston-website  
Description: Professional aerospace manufacturing website for Reliable Machine Boston
Visibility: Public
Do NOT initialize with README
Click "Create repository"

STEP 2: PUSH CODE TO GITHUB
Copy and paste these commands in terminal:

cd cloudflare-deploy
git init
git branch -M main
git add .
git commit -m "Initial commit - ReliableBoston.com website"
git remote add origin https://github.com/YOUR_USERNAME/reliableboston-website.git
git push -u origin main

STEP 3: CONNECT CLOUDFLARE PAGES TO GITHUB
1. Go to: https://dash.cloudflare.com → Pages
2. "Create a project" → "Connect to Git"  
3. Authorize Cloudflare to access GitHub
4. Select: reliableboston-website repository
5. Project name: reliableboston-com
6. Build settings: (leave all empty)
7. "Save and Deploy"

STEP 4: ADD CUSTOM DOMAIN (NO PROXY)
1. Pages project → Custom domains
2. Add: reliableboston.com (DNS only - gray cloud)
3. Add: www.reliableboston.com (DNS only - gray cloud)

STEP 5: DNS RECORDS AT REGISTRAR
Type: A      Name: @    Value: [IP from Pages]
Type: CNAME  Name: www  Value: reliableboston-com.pages.dev
